## CPU

cpu 是任何计算机系统的核心

### CPU Utilization

CPU 指标是监控系统性能的一个重要指标。

CPU 使用是一个描述 CPU 在某一段时间内有多少任务被处理，对于不同的任务，复杂度
不同，通常不好量化。

CPU 使用率通过一段指定的时间，把 CPU 主要分为两个状态（idle 和 非 idle）

CPU Utilization (U) = 100% – (Percentage of time spent in idle task)
